# Sons d'ambiance

Ce dossier doit contenir les fichiers audio (libres de droits) utilisés
par le module "Ambiances sonores de concentration" (§4 du cahier des
charges). Ils ne sont pas fournis automatiquement — à toi d'y déposer de
vrais fichiers `.mp3` avec exactement ces noms :

- `pluie.mp3`
- `bruit_blanc.mp3`
- `foret.mp3`
- `riviere.mp3`
- `vent.mp3`
- `musique_instrumentale.mp3`
- `bibliotheque.mp3`
- `cafe_etude.mp3`

Tant qu'un fichier n'est pas présent, l'app affiche un message d'erreur
si on essaie de le lire (au lieu de planter).
